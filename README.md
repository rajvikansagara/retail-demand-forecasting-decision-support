# retail-demand-forecasting-decision-support
Comparative analysis of machine learning and deep learning models for retail demand forecasting and inventory decision-support

## Overview

This repository contains the implementation developed for the MSc thesis entitled:

**Comparative Analysis of Machine Learning and Deep Learning Models for Retail Demand Forecasting and Inventory Decision-Support**

The project compares traditional statistical, machine learning, and deep learning models for retail demand forecasting and demonstrates how forecasting outputs can support inventory decision-making.

## Dataset

The study uses the Rossmann Store Sales dataset.

Files include:

- train.csv
- test.csv
- store.csv

## Models

- SARIMA
- XGBoost
- LSTM
